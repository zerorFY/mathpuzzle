# 数学填字 Puzzle Generator

一个纯静态网页版本的数学填字生成器。

## 当前一稿功能

- 从左上角数字节点开始扩散生成数学网络
- 每条算式都是固定 5 格：`数字 运算符 数字 = 结果`
- 运算符可选：`+ - × ÷`
- 结果上限可调，默认 `≤ 100`
- 网格结构可调：`13 x 13`、`17 x 17`、`21 x 21`
- 默认高密度生成：
  - `13 x 13` 默认最多 24 条算式
  - `17 x 17` 默认最多 40 条算式
  - `21 x 21` 默认最多 60 条算式
- 自动生成完整答案网格
- 统一简单挖空策略：
  - 只挖数字和运算符
  - 不挖等号
  - 每条算式最多挖 2 格
  - 空格可直接输入作答
  - 支持检查答案、清空填写
  - 支持显示/隐藏答案

## 本地打开

直接用浏览器打开 `index.html` 即可。
